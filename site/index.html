Atelier Apache Spark


Ce(tte) œuvre est mise à disposition selon les termes de la Licence Creative Commons Attribution - Pas d’Utilisation Commerciale - Partage dans les Mêmes Conditions 4.0 International.

Objectif

L'objectif de ce cours est d'initier les apprenants aux concepts de base de Apache Spark, et de faire le tours des composants qui le constituent et des cas d'utilisation. Ce cours est renforcé par des travaux pratiques.

Table des Matières

Le cours est divisé comme suit:

  1. Introduction aux Big Data
  2. Présentation de Apache Spark
  3. Installation de Spark
  4. Spark Batch
  5. Spark SQL
  6. Spark Streaming
  7. Spark ML-Lib
  8. Spark GraphX
